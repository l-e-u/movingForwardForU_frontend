# Parcel Manager

## About
This is the frontend of a business tool that helps with the organization and management of deliveries. It's a solution for a growing business that needs a way to create jobs and assign them to drivers. The main focus of the UI/UX is to be responsive and reduce manual input for safety and accuracy since drivers are always on the move.

* Uses React and CSS for styling.
* Responsive, mobile-first design.
* Simple, clean layout for legibility.
* Uses Google Maps API for address autocomplete.
